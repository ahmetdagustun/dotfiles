## 🚀 Quick Setup
1. Install Homebrew.
2. Clone this repo: `git clone https://github.com/ahmetdagustun/dotfiles.git ~/dotfiles`
3. Run `brew bundle --file=~/dotfiles/Brewfile`.
4. Run `chmod +x ~/dotfiles/setup.sh && ~/dotfiles/setup.sh`.
